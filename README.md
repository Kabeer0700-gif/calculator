# 🧮 Calculate Me! — Web Calculator

A sleek, interactive web-based calculator built with vanilla HTML, CSS, and JavaScript. Deployed live on Vercel and supports standard arithmetic along with binary and octal number system conversions.

🔗 **Live Demo:** [calculator-indol-nu-86.vercel.app](https://calculator-indol-nu-86.vercel.app)

---

## 📁 Repository Structure

```
calculator/
│
├── index.html      # Main HTML structure and button layout
├── style.css       # Styling and layout
├── script.js       # Calculator logic and number conversion
└── README.md
```

---

## ✨ Features

- **Basic Arithmetic** — Addition, Subtraction, Multiplication, Division
- **Decimal Support** — Floating point calculations
- **Quick Input** — One-tap button for the value `100`
- **Binary Conversion** — Convert the current value to Binary (Bin)
- **Octal Conversion** — Convert the current value to Octal (OCT)
- **Clear Display** — AC button resets the calculator instantly
- **Read-only Input Field** — Prevents manual typing errors; all input is button-driven
- **Responsive Design** — Clean, styled layout that works across devices

## 🚀 Getting Started

### Run Locally

No installation required — just open the file in your browser:

```bash
# Clone the repository
git clone https://github.com/Kabeer0700-gif/calculator.git

# Navigate into the folder
cd calculator

# Open in browser
open index.html
```

Or simply double-click `index.html` to launch it in your default browser.

---

## 🔢 Number System Conversions

| Button | Function |
|---|---|
| `Bin` | Converts the current displayed number to **Binary** |
| `OCT` | Converts the current displayed number to **Octal** |
| `AC` | Clears the display |
